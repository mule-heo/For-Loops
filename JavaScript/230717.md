merge(person , address)  두 개의 객체를 병합하는 함수를 만들어주세요.

```
var person = {
    name : 'John',
    age  : 24
}

var address = {
    addressLine1 : 'Some Location x',
    addressLine2 : 'Some Location y',
    city : 'NewYork'
}

merge(person, address) 
/* name , age , addressLine1 , addressLine2 , city */
```
