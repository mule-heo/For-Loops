# 8월 22일 화요일 & | - 타입의 교집합과 합집합

1. & | 타입의 교집합과 합집합

   ```
   type OrderStatus = "pending" | "completed";
   type DeliveryStatus = "completed" | "shipped";
   type Status = OrderStatus | DeliveryStatus;
   ```

    다음 Status 타입 결과를 알려주세요.

    ```
    type Status = OrderStatus | DeliveryStatus;
    ```

    다음 Status 타입 결과를 알려주세요.

    ```
    type Status = OrderStatus & DeliveryStatus;
    ```
